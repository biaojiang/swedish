Swedish Learning

Introduction

This is a new Swedish learning online book compiled with mdBook: a utility to create modern online books from Markdown files with the modern high-performance language Rust. The previous book made with Python Sphinx will be archived.

Another more advanced book is available at Avancerade svenska ord, fraser och grammatik.

What this book contains

  • Curated vocabulary lists from SVT, DN, Netflix, and other sources.
  • Each list includes Swedish definitions and example sentences, plus English/Chinese explanations when available.
  • A growing glossary and index for quick lookup.

How to use this book

  • Start with the latest SVT or DN month and read the tables under “Ord”.
  • Use the glossary and index to review and cross-check repeated terms.
  • Revisit older months to reinforce words that reappear.

Build and view locally

If you have mdBook installed:

mdbook serve

Then open the local URL that mdBook prints in your browser.

📌 What is a particle verb?

In Swedish (like in English), some verbs come with a small word — called a particle — that changes the meaning of the verb.

For example:

Verb+ ParticleMeaning
gå (to go)åtgo towards / be used up → gå åt
ta (to take)påto touch → ta på
komma (to come)förbito drop by → komma förbi

These particles look like prepositions (like “på”, “åt”, “förbi”), but they’re not acting as prepositions here — they’re just part of the verb.

1. “andre” vs “andra”

Both come from andra = “second / other”, but the form changes depending on grammar.

  • den andra → the other (one) (neutral/general)
  • den andre → the other (one) (masculine, specific person)

🔹 Regel

När du har den/det/de + adjektiv + substantiv, så:

  • adjektivet får -e (eller ibland -a i plural)

Ex:

  • den misstänkte skytten → the suspected shooter

  • den gripne mannen → the arrested man

  • den dömde brottslingen → the convicted criminal 👉 Här är uttrycket:

  • få reda på = to find out / to learn / to discover

🔹 Vad betyder reda?

Själva ordet reda betyder ungefär:

  • order / clarity / knowing what’s going on

Men i praktiken används det nästan alltid i fasta uttryck.

🔹 Viktiga uttryck med reda

  • ta reda på → find out (aktivt)
    Jag ska ta reda på det = I’ll find out
  • få reda på → find out (något blir känt)
    Jag fick reda på sanningen = I found out the truth
  • ha reda på → keep track of / know
    Har du reda på var han är? = Do you know where he is?
